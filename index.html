Projects

2017 Handy
Handy is a body-language processor that I co-developed with Spencer Gilbert at WildHacks 2017. Handy uses the Leap Motion API to track the hand and finger movements of the speaker, providing an analysis of the speaker's body language performance. Feedback includes analysis and visualization of hand speed, placement, orientation, and finger states throughout the speech.

Technologies used: Python, Leap Motion SDK, Javascript, Flask
2017 Endangered Planet
Endangered Planet is an educational mobile app intended to develop an accessible way to create awareness and avenues for aid to threatened species across the world. Specific species information ranging from population trends to geographical location are provided in each species' profile. As of version 1.2, species can be added to a favorites list which is accessible from the main screen. Data is provided from the IUCN Red List API. This app has accumulated more than 100 installations since its release.

Technologies used: Swift, XCode, MapKit
2016 SatActbot
SatActbot is a conversion bot that converts between SAT and ACT scores based on the Concordance Tables provided by Collegeboard. It was built with Reddit's API and currently serves 4 different subreddits related to college admissions. It has been referenced over 800 times and has accumulated 200+ comment karma.

Technologies used: Python, PRAW, Heroku
2016 LeapDIVE
LeapDIVE, developed at Duke University's Virtualization Lab in the summer of 2016, was my first experience with the Oculus VR headset, Leap Motion controller, and a CAVE-type system. Within the virtual world, the user can pick up and interact with objects with their hands instead of the traditional touch controller.

Technologies used: Unity, C#, Oculus Rift, Orion SDK, Leap Motion SDK
2016 CalcApprox
Created for my AP Calculus final project, CalcApprox parses single-variable functions and integrates them using Simpson's Rule and Trapezoidal Rule. It also produces a graph displaying the size of error from each method by interval length against the result from actually integrating the function.

Technologies used: Python, Matplotlib, SymPy
